# IMDB Movie Recommender

## Description
A content-based movie recommendation system built using Python and machine learning techniques. The app suggests similar movies based on plot descriptions from the IMDB Top 1000 dataset.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit

## How It Works
- Uses TF-IDF vectorization on movie plot summaries
- Computes cosine similarity to find similar movies
- Interactive Streamlit interface for user input
